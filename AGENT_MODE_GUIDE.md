# Agent Mode Guide

## What is Agent Mode?

Agent Mode transforms the SEO assistant from a responsive tool into a proactive strategic partner. Instead of just answering your questions, it thinks deeply about your SEO challenges and guides you through a comprehensive strategy.

## Key Differences: Ask vs Agent Mode

### Ask Mode (Default)
- **You're in control**: You tell it exactly what to do
- **Direct responses**: Quick, focused answers to specific questions
- **Command-based**: "Research keywords for X", "Analyze example.com"
- **Best for**: Specific tasks, quick lookups, experienced SEO practitioners

### Agent Mode (Strategic)
- **AI-guided workflow**: It takes initiative and plans the approach
- **Strategic thinking**: Deep analysis with chain-of-thought reasoning
- **Opinionated advice**: Data-driven recommendations on what to prioritize
- **Best for**: Building strategies, comprehensive site audits, learning SEO

## What Makes Agent Mode Different?

### 1. Chain-of-Thought Reasoning
Every response includes hidden strategic analysis:
- Situation assessment (what you're trying to achieve)
- Strategic considerations (opportunities and challenges)
- Recommended approach (why this path makes sense)
- Next steps (how to move forward)

### 2. Strategic Framework
Agent mode doesn't just list keywords - it builds a framework:
- **Content Pillars**: 3-5 main topical themes
- **Keyword Clusters**: Supporting keywords grouped by intent
- **3-Tier Priority System**:
  - Tier 1 (Quick Wins): Target these NOW
  - Tier 2 (Authority Building): Build topical authority
  - Tier 3 (Long-term): Aspirational competitive terms

### 3. Opinionated SEO Guidance
Agent mode has strong opinions backed by data:
- Volume isn't everything (intent matters more)
- Content clustering beats thin pages
- Match content format to search intent
- Don't chase impossible keywords early
- Quality content > keyword stuffing

### 4. Competitive Intelligence
Goes beyond just showing data:
- Analyzes who's ranking and why
- Identifies content gaps competitors miss
- Suggests specific angles to differentiate
- Estimates realistic ranking timelines

## Common Questions

**Q: Can I switch between modes mid-conversation?**
A: Yes! The mode selector is always available. Agent mode has full conversation context.

**Q: Does agent mode cost more?**
A: No, it uses the same API calls. It's just a different prompting strategy.

**Q: Is agent mode slower?**
A: Slightly - it does more strategic thinking and often uses multiple tools automatically.

**Q: Can I use agent mode for quick questions?**
A: You can, but Ask mode is more efficient for quick lookups.

**Q: Will agent mode always do full analysis?**
A: It adapts to your request. If you ask a simple question, it gives a simple answer. 
   If you ask for strategy, it provides comprehensive analysis.
