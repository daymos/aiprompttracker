## Key Concepts:

- **Q* (Quality Signal)**: Site-level quality scoring
- **P* (Popularity Signal)**: User interaction data (Navboost)
- **E-E-A-T**: Experience, Expertise, Authority, Trustworthiness
- **Entity SEO**: Building recognition as a known entity
- **Zero-Click Marketing**: Optimizing for SERP features
- **User Signals**: Clicks, dwell time, engagement metrics
