# KeywordsChat Frontend

Flutter web app for conversational keyword research.

## Setup

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Configure backend URL in `lib/services/api_service.dart` and `lib/services/auth_service.dart`

3. Run in Chrome:
```bash
flutter run -d chrome
```

4. Build for production:
```bash
flutter build web
```

## Features

- Google Sign-In authentication
- Chat-based keyword research
- Conversation history
- Responsive design
- Dark mode
